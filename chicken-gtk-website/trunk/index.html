Chicken Gtk

The chicken gtk project plans to make bindings for the chicken scheme compiler. Right now I plan to use swig to generate the basic "C" functions. Later I will wrap them in a "nice" prometheus object sytem. In the future I might use another system for generating the basic "C" code. Once the gtk bindings are created I plan to wrap gstreamer.
