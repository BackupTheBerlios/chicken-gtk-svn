Building chicken gtk binding from source

Right now these instructions only work for linux. With some modification it should be possible to build under windows with the mingw compiler. This is alpha software it has not been tested.

Requirements

  • Chicken Scheme Compiler 2.3+ with these extensions installed
    • prometheus
  • Plt Scheme Compiler or more specifically MzScheme. Used for generating the code for bindings. (In future maybe I will change it so chicken can be used)
  • Scons 0.96.91 I use that version, other versions might work.
  • Gtk dev libs. For libglib,libgtk etc :)
